2. item: flex-grow

        flex-grow : container의 크기 증감에 따른 item의 크기 증가 비율을 지정하는 속성
        * flex-grow:0 => container의 크기가 증감해도 item의 크기가 변하지 않는다.
        * flex-grow: 1 이상 => container를 가득 채우도록 item의 크기가 결정됨

3. flex-shrink: container의 크기감소에 따른 item의 크기감소 비율을 지정하는 속성 

      * flex-shrink:0 => container의 크기가 감소해도 item의 크기가 감소하지 않음
      * flex-shrink:1 이상 => 크기감소에 따른 감소폭을 지정

4. flex-basis: item의 기본 크기를 결정하는 속성, default, auto

      * flex-basis => item의 크기를 지정할 때 모든 단위를 사용할 수 있음
      * flex-basis => 속성값 0으로 지정하면 absolute flex item이 되어서 content의 크기로 item의 크기를 결정
