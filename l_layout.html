block / inline element의 영역과 크기

1. block / inline element의 영역

block 요소는 너비가 부모요소의 너비와 동일하고,
높이는 콘텐츠의 높이와 동일하다.
block 요소는 너비가 부모요소의 너비와 동일하고,
높이는 콘텐츠의 높이와 동일하다.
inline 요소는 너비가 콘텐츠의 너비와 동일하고
높이는 콘텐츠의 높이와 동일하다.

블록과 인라인의 크기

블록요소는 width와 height속성을 사용가능
inline요소는 width와 height를 사용할 수 없다. 
      margin상쇄 : 두 요소의 margin-top과 margin-bottom이 인접할 경우 두 요소 중 마진값이 큰 요소의 마진만 적용
